# Resqio Style Guide: Wortwahl & Tonalität

Dieses Dokument dient als Leitfaden für die Sprache auf der Resqio Website und in der Dokumentation. Ziel ist es, eine professionelle, kameradschaftliche und authentische Sprache zu verwenden, ohne dabei rechtliche Versprechen oder Zertifizierungen zu suggerieren, die nicht formal vorliegen.

## 1. Grundprinzipien
- **Authentisch statt bürokratisch:** Wir sprechen die Sprache der Feuerwehr.
- **Unterstützend statt garantierend:** Wir helfen bei der Einhaltung von Standards, garantieren aber keine formale "Rechtssicherheit" oder "Zertifizierung".
- **Fokus auf Nutzwert:** Was bringt es dem Gerätewart/Kommandanten im Alltag?

## 2. Wortwahl (Darf -> Sollte)

| "Hartes" Wort (Vermeiden) | "Weiches" Wort (Nutzen) |
| :--- | :--- |
| Rechtssicher / Rechtssicherheit | Nachvollziehbar, Lückenlos, Dokumentiert, Prüfsicher (im Sinne der Dokumentation) |
| Konform / Rechtskonformität | Standardbasiert, Praxisnah, Strukturiert, Regelkonform (vorsichtig nutzen) |
| Zertifiziert | Geprüft, Bewährt, Etabliert |
| Compliance | Verantwortungsbewusst, Transparenz, Ordnung |
| Gesetzlich | Regulär, Fristgerecht, Anforderungsgemäß |
| 100% DSGVO-konform | Datenschutz-fokussiert, Sicher gehostet in DE, Privatsphäre-bewusst |
| Verpflichtend | Notwendig, Wichtig, Empfohlen |

## 3. Feuerwehr-Vokabular (Bevorzugt)
- **Mannschaft** -> Personal & Kameraden
- **Ausrüstung** -> Gerätehaus & Technik
- **Wartung** -> Technischer Dienst / Prüfung
- **Übung** -> Ausbildung & Dienst

## 4. Haftungsausschluss & AI-Hinweis
- Wir kommunizieren offen (aber dezent), dass Bilder/Texte KI-unterstützt erstellt wurden.
- Wir betonen, dass die Software ein *Hilfsmittel* ist, die letzte Verantwortung aber beim Nutzer liegt.
