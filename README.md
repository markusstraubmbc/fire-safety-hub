# RESQIO Fire Safety Hub

Marketing-Website for the RESQIO fire department management platform.

## Tech Stack

- Vite
- TypeScript
- React
- shadcn/ui
- Tailwind CSS

## Development

```bash
# Install dependencies
npm i

# Start dev server (port 8080)
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview

# Lint
npm run lint
```

## Deployment

Production builds are output to `dist/`. The site is a static SPA suitable for any static hosting provider.
